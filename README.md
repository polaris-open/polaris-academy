# Polaris Academy

> **Status: draft v0.1.** This repository is not a complete course yet.

Polaris Academy is the **open source educational area of Polaris**: self-paced,
project-based and anti-hype learning for software engineering, applied AI
engineering and technical growth.

This repo is in an early, honest draft state. It exists to prove the **first 30
minutes** of a self-taught learner — not to deliver a finished curriculum.

## What this is

- A growing collection of **formations** (structured learning paths) built around
  **real, buildable projects** instead of hype.
- The first formation is **Applied AI Engineering**.
- A minimal **starter** so you can begin in 30 minutes, with no LLM, no cloud and
  no paid API.

## What this is not (yet)

- Not a complete course.
- Not a certification. It does not promise that you become a "specialist".
- Not an LMS or a platform. The canonical document is the architecture, **not the
  course**.

## Principles

- **Self-paced first.** Cohort optional. Mentor optional. Review optional.
- **Project-based learning.** You build one system that grows.
- **Community is support, not dependency.** The path must work even if you study
  100% alone.
- **Synthetic data only.** All public learning data must be synthetic — never real
  production logs, customer data, corporate data, secrets, API keys, credentials or
  real PII.
- **Demos are not production.** "It worked once" is never "it is ready".
- **Specialist is a direction of maturity, not a title promise.**

## What works today

- The Applied AI Engineering formation architecture.
- Level 1 entry path.
- Lab 01 and Lab 02.
- The AI Incident Support Assistant starter.
- Synthetic dataset validation.
- Manual dataset inspection.
- Weak deterministic baseline.
- Minimum evidence checking.

## What does not work yet

- Level 2 materials are planned, not built.
- No LLM, RAG or Agent is implemented.
- No certification or LMS exists.
- No production-ready AI system is provided.

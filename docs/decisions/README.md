# Decisions

> **Status: draft v0.1.**

Relevant decisions about Polaris Academy and its formations will be recorded here as
**ADRs (Architecture Decision Records)** and decision logs — one decision per file,
short and honest.

Use the [ADR template](../../formations/applied-ai-engineering/templates/adr-template.md)
as a starting point. Each ADR should capture the context, the options considered,
the decision, its consequences, and the trigger that would make you reverse it.

The architectural decisions already made for the first formation are consolidated in
the canonical document (Section 16), at
[../../formations/applied-ai-engineering/canonical/](../../formations/applied-ai-engineering/canonical/).

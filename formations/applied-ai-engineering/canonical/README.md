# Canonical document

> **Status: draft v0.1.**

This folder holds the **canonical document** of the Applied AI Engineering
formation:

- [formacao-polaris-applied-ai-engineering-v0.1.md](formacao-polaris-applied-ai-engineering-v0.1.md)

## What the canonical document is

It defines the **architecture, patterns, levels, rubrics and criteria** of the
formation. It is the single source of truth for *how the formation is structured*
and *what "good" means*.

## What it is not

**It is not the complete course.** It is not lesson material. It should not be
turned into a full course. It is the blueprint that the real teaching materials are
built against.

## What comes later

The real didactic materials are built **after** this document, level by level, and
validated with real learners:

- module pages
- lab guides
- troubleshooting guides
- FAQs
- templates
- starter repos
- reference solutions
- examples

Until those exist, self-study works with the **starter + self-assessment +
machine-checkable blockers**, and the calibration is **partial by design**. That
limitation is stated openly in the canonical document itself (Section 18).

## Editing rule

The canonical document is copied here verbatim. Do not rewrite, summarize,
translate, reorganize, change headings, change examples or change links in place.
Changes to the architecture should go through a new versioned revision and a
decision log (see [../../../docs/decisions/README.md](../../../docs/decisions/README.md)).

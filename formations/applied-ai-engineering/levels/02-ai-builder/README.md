# Level 2 — AI Builder

> **AI Builder — planned.**
>
> **Status: draft v0.1.** Materials for this level have not been built yet.

This level takes you from "I can use ChatGPT" to "I can **build** with an LLM in a
**measured** way": APIs, engineering-grade prompting, structured outputs with
validation, cost/latency, and the core idea — **evals from day one**. In parallel:
classic ML at a minimum and embeddings as a **bridge to RAG**.

The full architecture (modules, milestones, labs, anti-patterns, rubric, blockers)
is defined in the canonical document, Section 10:

- [../../canonical/formacao-polaris-applied-ai-engineering-v0.1.md](../../canonical/formacao-polaris-applied-ai-engineering-v0.1.md)

The didactic materials (module pages, lab guides, troubleshooting, FAQs, reference
solutions) will be built later, after Level 1 is validated with real learners.

> Reminder: **semantic search is not RAG**, and an LLM with no eval is not a
> finished feature. Start at [Level 1](../01-ai-foundations-for-builders/README.md).

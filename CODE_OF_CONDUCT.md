# Code of Conduct

> **Status: draft v0.1.**

Polaris Academy is an open, educational, and welcoming community. We expect everyone
who participates — learners, contributors and maintainers — to act with **respect**.

The organization-wide Code of Conduct lives in the shared
[`polaris-open/.github`](https://github.com/polaris-open/.github) repository and
applies here. This file is a short pointer, not a separate policy.

## In short

- **Be respectful.** Assume good faith. Critique ideas, not people.
- **Be helpful.** This is a learning space; people are at different levels.
- **Keep it safe.** Synthetic data only — never share secrets, real PII, customer
  data or corporate data.

If you experience or witness unacceptable behaviour, follow the reporting process in
the organization-wide Code of Conduct.
